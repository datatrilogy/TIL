### ✶주의사항✶ 
    1. **현재위치 어딘지 항상 확인**
    2. **Repo 안에 Repo(master)를 만들지 않는다.**
       1. 이미 git init을 실행한 폴더안에 
       2. 또 다른 폴더를 만들고
       3. 그 폴더 (하위폴더)에서 `git init`을 하지 않는다. 
       4. Master 떠있으면 init 제발 X
    3. **Home에서 init 하지 않는다.**
    4. **지금은 Github에서 직접 수정하지 않는다.**
